# bicycle-charging-server
基于SpringBoot3+Vue3前后端分离的电动车充电服务网站，系统包含管理员、维修员、用户角色，功能如下： 管理员：首页统计、用户管理、品牌管理、站点管理、电桩管理、充电管理（充电预约、订单管理、站点评价）、卡券管理（卡券管理、卡券领取）、报修管理、政策新闻、系统留言、公告信息。 维修员：站点查看；电桩查看；报修管理（处理结束）；个人信息修改；密码修改。 用户：注册、登录、首页、充电站（站点详情、站点收藏、站点评价、充电桩列表、充电桩预约）、卡券活动（卡券详情、领取）、政策新闻、系统留言、系统公告。用户还具有个人中心功能，功能包括：站点收藏、充电预约（开始充电）、我的订单（结束充电、缴费，可使用卡券）、站点评价、我的卡券、设备报修、系统留言、个人信息、修改密码。
